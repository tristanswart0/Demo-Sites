# Dark Brew Cafe - Example Website

A sleek, dark-themed example website for a premium coffee cafe. This project showcases modern web design techniques with a focus on user experience and responsive layout.

## Features

- **Sleek Dark Design**: Modern dark theme with elegant typography and smooth animations
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Three Pages**: Home, Menu, and About pages with comprehensive information
- **Professional Content**: Detailed menu, company story, team information, and contact details

## Pages

1. **Home (index.html)**: Hero section, featured drinks, about preview, and location info
2. **Menu (menu.html)**: Complete drink and food menu with pricing
3. **About (about.html)**: Company story, values, team members, and roasting process

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Responsive Design
- Modern Font Stacks

## Local Development

To view the website locally:

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Or use a local server (recommended for better experience)

### Using Python (if available)

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`
